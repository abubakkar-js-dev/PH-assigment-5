Important FAQ

What is the Document Object Model (DOM)?

The DOM is like a tree structure that represents the content of a web page. It allows programming languages (like JavaScript) to change the content, structure, and style of the page dynamically.

How do you select an element from the DOM?

We can select elements in the DOM using JavaScript methods like:

  • getElementById() – Selects an element by its ID.
  • getElementsByClassName() – Selects elements by their class name.
  • querySelector() – Selects the first element that matches a CSS selector.
  • querySelectorAll() – Selects all elements that match a CSS selector.
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is when we add one event listener to a parent element instead of putting separate listeners on each child element. This way, the parent handles events for all its children making it easier to manage, especially when new child elements are added.

How do you manipulate an element's attributes and styles using the DOM?

We can change an element’s attributes (like src, href, etc.) and styles using JavaScript:

  • To change an attribute, use setAttribute().
  • To change styles, use element.style, like element.style.color = 'red' to change text color.
